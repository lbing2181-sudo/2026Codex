# Tool Stack

## Objective

Define a practical production stack for building `Sable Archive` without losing continuity, brand control, or platform fit.

## Core Production Principle

Images come first.
Video comes second.

The first phase of the account is for:

- locking the visual canon
- locking recurring women
- locking wardrobe grammar
- locking the realm atmosphere
- building thumbnail recognition on X

Video should amplify an established canon, not define it.

## Recommended Media Mix

For the first 4 to 8 weeks:

- 70% still images
- 20% short motion pieces
- 10% longform text or archive posts

## Tool Roles

### 1. FLUX.2

Primary role:

- character canon locking
- repeatable portrait generation
- wardrobe iteration
- continuity-preserving edits
- season-one still production

Why:

- best fit for reusable structured prompts
- strong candidate for multi-reference workflows
- most suitable for treating recurring characters as assets rather than disposable outputs

### 2. Midjourney

Primary role:

- taste exploration
- moodboarding
- fabric, ornament, and realm atmosphere exploration
- visual direction testing

Why:

- useful for searching for the right air, texture, and silhouette language
- less ideal as the sole canon engine for recurring character consistency

### 3. Runway Gen-4

Primary role:

- short motion derived from locked stills
- gentle cinematic movement
- atmosphere amplification
- teaser-style clips for X

Why:

- best used after the still image already works
- ideal for veil movement, slow turns, ambient light, smoke, and restrained camera drift

### 4. Sora or Veo

Primary role:

- second-phase archive films
- richer short narrative pieces
- more advanced realm trailers or dossier clips

Why:

- useful once the canon is visually stabilized
- should not replace still-image canon building at the opening stage

## Production Sequence

1. generate canonical stills
2. select only the most authored and recognizable outputs
3. lock recurring characters into dossiers
4. publish stills first
5. convert selected stills into short motion only after continuity holds

## Prompt Guidance By Tool

### For FLUX.2

- use layered structured prompts
- foreground identity anchors
- reference prior canon when recurring figures are involved

### For Midjourney

- use concise aesthetic direction
- emphasize mood, material, silhouette, and environment
- do not trust it alone for recurring character identity

### For Runway

- let the input image carry identity and composition
- use text mainly for motion behavior
- prefer restrained movement over spectacle

### For Sora or Veo

- treat them as phase-two narrative tools
- use after character, wardrobe, and environment canon have already been established

## Quality Filter

Reject outputs that feel like:

- generic AI luxury fantasy
- NFT-style logo work
- over-rendered game splash art
- symmetrical but forgettable beauty
- motion that adds energy without adding identity
