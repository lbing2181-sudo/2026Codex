# Platform Matrix

## Objective

Define how `Sable Archive` should expand beyond X without losing authorship, continuity, or brand dignity.

## Core Rule

Do not run a full matrix too early.

The matrix should be staged, not simultaneous.
The main risk is not missing reach.
The main risk is diluting the canon before the audience remembers it.

## Strategic Conclusion

- X is the primary worldbuilding platform.
- Instagram is the secondary visual showroom.
- TikTok is the delayed motion amplifier.

## Why This Order

### X

Best use:

- archive fragments
- character dossiers
- serialized canon
- pinned gateway post
- longform Articles
- audience interpretation and quote-post behavior

Role:

`the world`

### Instagram

Best use:

- strongest still images
- carousel sequences
- polished brand-facing presentation
- mood-driven reels after canon exists

Role:

`the gallery`

### TikTok

Best use:

- short atmospheric motion
- teaser edits
- seductive but non-explicit movement
- ritual, smoke, veil, lantern, and camera drift pieces

Role:

`the trailer house`

## Rollout Stages

### Stage 1: X First

Focus:

- fully operate X
- establish the archive identity
- build the first season memory

Instagram:

- secure the handle if available
- mirror the brand identity
- publish lightly or not at all

TikTok:

- secure the handle if available
- do not force early output

### Stage 2: Instagram Activation

Trigger:

- X has a stable profile identity
- at least 15 to 25 consistent canon assets exist
- the first season visual grammar is recognizable

Focus:

- post the cleanest stills
- use carousel logic for mini-dossiers
- keep captions shorter than X

### Stage 3: TikTok Activation

Trigger:

- still-image canon is stable
- several strong stills have proven motion potential
- the project has a repeatable short-video language

Focus:

- 5 to 12 second motion assets
- still-derived micro-cinematics
- no heavy lore explanation

## Adaptation Rule

One mother asset should become three platform-specific expressions.

Example:

- X: still image + lore line + hook
- Instagram: still image or carousel + restrained caption
- TikTok: short motion derived from the same canon asset

This is not reposting.
This is controlled adaptation.

## What Not To Do

- do not post identical captions everywhere
- do not launch all three platforms as if they have equal weight
- do not let TikTok shape the canon too early
- do not let Instagram become a generic beauty feed
- do not let X collapse into random image posting

## Opening Recommendation

Current best move:

- operate X as the true main channel
- secure Instagram and TikTok handles now if possible
- keep Instagram light
- delay TikTok until the motion layer is ready

## Success Standard

The matrix is working when:

- each platform has a clear role
- the same universe remains legible everywhere
- the audience feels one archive, not three disconnected content accounts
